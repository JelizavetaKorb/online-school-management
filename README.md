# Online School Management System

A full-stack web application designed to manage tutors, students, lessons, and progress tracking for an online school environment. Built for flexibility, usability, and real-world school administration scenarios.

## Key Features

- **User Management**
  - Register and manage **students** and **tutors**  
  - Assign students to tutors and track their learning progress 

- **Lesson Scheduling & Progress Tracking**
  - Schedule lessons and manage availability
  - Track student progress and completed lessons

- **Full-Stack Functionality**
  - Backend: C#, .NET, Microsoft Entity Framework (EF)
  - Frontend: HTML, CSS, JavaScript, Razor Pges
  - Database: SQL via EF 

- **Data Management**
  - CRUD operations for students, tutors, lessons
  - Secure and consistent storage of user data

- **User Interface**
  - Clean, responsive web design
  - Easy navigation for both students and tutors

- **Extensibility**
  - Designed for easy addition of new features
  - Modular architecture for maintainable code

## Technologies Used

- **Backend:** C#, .NET, Entity Framework (EF)  
- **Frontend:** JavaScript, HTML, CSS  
- **Database:** SQL (via EF)  
- **Development Tools:** Rider, Git
